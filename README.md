# Pup Dash 🐶 — Luna vs Dolly

A mobile-first, player-vs-computer dog racing game in a single HTML file. **Luna** the beagle
takes on **Dolly** the Weimaraner over a 250m park sprint — pick your pup on the title screen
and the other one races you.

Open [index.html](index.html) in a browser (best on a phone in portrait). Drum the 🐾 pad fast
to run — every tap counts. ⚡ fires ZOOMIES when the meter is full; 🦴 bones restore stamina.
Three difficulties; best times are saved on your device.

Built on the Dachshund Dash engine with per-breed body shapes: no assets, no build step,
canvas + WebAudio only.
